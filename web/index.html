Audio Streamer

Streaming Services

All services run simultaneously. Play from any source — switching is seamless.

AirPlay 1 (classic) — Ceòl Stream uses AirPlay 1 deliberately. AirPlay 1 streams truly lossless audio (Apple Lossless / ALAC at 16-bit / 44.1 kHz, CD quality). AirPlay 2, despite Apple's claims, uses lossy AAC encoding in many scenarios. For bit-perfect audiophile playback, AirPlay 1 is the correct choice.

USB DAC

Select your USB DAC for audio output. Signal path: Network → Service → ALSA → USB DAC (bit-perfect).

Bit-perfect playback — No sample rate conversion, no mixing, no volume adjustment in software. Each service sends audio directly to the ALSA hardware device. The DAC receives the original signal untouched. Roon and HQPlayer support up to 24-bit / 192 kHz. AirPlay streams at 16-bit / 44.1 kHz (CD quality, lossless).

DSD support — Roon Bridge and HQPlayer NAA support DSD streaming (DoP) to compatible DACs. AirPlay does not support DSD. Check each DAC's capabilities above to see if your device supports DSD.
